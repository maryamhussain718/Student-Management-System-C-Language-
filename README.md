
# Student Management System (C Language)

## Project Description

Student Management System is a console-based application developed in C language. It helps users manage student records using file handling. The program allows users to add, view, search, update, and delete student records through a simple menu-driven interface.

## Features

* Add New Student
* View All Students
* Search Student by ID
* Update Student Information
* Delete Student Record
* Store Records in a Binary File
* Duplicate Student ID Validation

## Technologies Used

* C Programming Language
* Structures
* Arrays
* Functions
* File Handling

## File Used

* `students.dat` – Stores all student records.

2. View All Students
3. Search Student
4. Update Student
5. Delete Student

## How to Run

1. Open the project in Code::Blocks, Dev-C++, or any C compiler.
2. Compile the source file.
3. Run the program.
4. Select an option from the menu.
5. Student records will be saved automatically in `students.dat`.
